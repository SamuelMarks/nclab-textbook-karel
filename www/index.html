For Contributors

The textbook is open to contributors. If you like to contribute with a bugfix, example, section or chapter - you are very welcome. Every contributor automatically becomes a co-author. If you like to contribute, please contact Pavel Solin with a brief description of your idea. You should be able to use Latex and Git (Github).
